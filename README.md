# Market Discounts Scraper
This project scrapes product discount data from two markets: **2nabiji.ge** and **Nikora Supermarket**.  
The script exports the results into an Excel file with a clean, structured format.

---

## Requirements
Install necessary Python dependencies:

```
pip install -r requirements.txt
```

Main packages used:
- selenium  
- pandas  
- openpyxl

---

## ChromeDriver Setup
1. Download **[ChromeDriver](https://googlechromelabs.github.io/chrome-for-testing/)** matching your local Chrome version and operating system. Check your Chrome version by opening Chrome and typing **chrome://version** in the address bar.  
Use this version to download the corresponding **ChromeDriver**. 
2. Place the ChromeDriver executable anywhere on your system.  

3. In `market_discounts.py`, update the `executable_path` variable with the location of your ChromeDriver:

```python
executable_path = "your/driver/path/chromedriver"
```

4. Make sure the ChromeDriver file has the proper permissions to be executed.

## Features
- Automated Selenium ChromeDriver configuration  
- Scrapes:
  - product names  
  - discounts  
  - old and new prices  
  - product links  
  - product images (2nabiji)  
- Infinite scroll handling for 2nabiji  
- Pagination handling for Nikora  
- Cleans and formats text  
- Saves the results into an Excel file with two sheets

---

## Running the Scraper
Run using Python directly:

```
python market_discounts.py
```

Output file:

```
market_discounts.xlsx
```

Sheets created:
- 2nabiji  
- nikora
